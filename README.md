# Arduino-SmokeSensor
Arduino-Smoke Sensor
## Pin connection
- Oled --> Arduino Nano
  -  VCC --> 5V
  -  GND --> GND
  -  SCL --> A5
  -  SDA --> A4

- MQ-3 Sensor --> Arduino Nano
  - VCC --> 5V
  - GND --> GND
  - A0 --> A1
 
- MQ-7 Sensor --> Arduino Nano
  - VCC --> 5V
  - GND --> GND
  - A0 --> A2

- MQ-2 Sensor --> Arduino Nano
  - VCC --> 5V
  - GND --> GND
  - A0 --> A0
 
  ## Now upload this code and check data on the Oled Display
